Detailed Description

A base abstract class for all the users commands.

This class represents an interface for every command available to the user.

virtual void Trentino::Console::Command::acceptOutputVisitor ( OutputVisitor &  consoleVisitor) [pure virtual]

Accept an output subsytem to print execution results The output facilities are designed according to the visitor pattern. So each command has its own visitor implementation for the specified output type (console, GUI, log). Executing this method the output system takes control, grabs outcome information from command private data and sends it toward the final output.
