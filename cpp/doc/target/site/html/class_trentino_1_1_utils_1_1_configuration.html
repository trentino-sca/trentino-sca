void Configuration::parse ( )

Calls necessary boost::program_options functionalities in order to parse arguments from the command line and a configuration file.

Exceptions:
std::runtime_errorIndicated an error in reading the configuration file
Attention:
The method throws also all the exception defined in boost::program_options. In particular boost::progrm_options::unknown_argument is thrown whenever an unregistered argument is encountered on the command line or in the configuration file.
