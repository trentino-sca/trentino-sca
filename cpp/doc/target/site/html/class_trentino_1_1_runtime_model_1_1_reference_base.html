Detailed Description

Represents an SCA reference at runtime.

The RuntimeModel::Reference class points to the corresponding information from the SCA Model representing an sca reference and provides additional information associated with a reference. Examples for such additional information are:

  • the reference state
  • collection of wires connected to the reference
