Detailed Description

A factory class used by Binding provider to register a new Binding to the Binding Framework a Factory actually describes the binding to be created by giving its ID A Factory can create a new Binding based on the binding context the Binding context is provided by the Binding framework The framework read context from a central configuration file of trentino
