Detailed Description

Represents an SCA reference at runtime.

The RuntimeModel::Reference class points to the corresponding SCA::Model::ComponentReference class and provides additional information associated with a reference. Examples for such additional information are:

  • the reference state
  • collection of wires connected to the reference
