Detailed Description

Represents an SCA service at runtime.

The RuntimeModel::ServiceBase class points to the corresponding information from the SCA Model representing an sca service and provides additional information associated with a service. Examples for such additional information are:

  • the services state
  • collection of wires connected to the service

enum Trentino::RuntimeModel::ServiceBase::State

The lifecycle of a Service starts when the component hosting the service is not in the unresolved state, otherwise, the service does not exist at all.

Enumerator:
Available 

Service is in the Service Registry. Service is accessible : component is not UNRESOLVED further conditions such as:

o Allow remote connection to services (start servers) o …

Unavailable 

Service is not in the registry or service does not fullfill the conditions to be available.
