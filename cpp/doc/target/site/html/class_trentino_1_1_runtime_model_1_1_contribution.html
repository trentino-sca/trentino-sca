Detailed Description

Represents an SCA contribution at runtime.

The RuntimeModel::Contribution class points to the corresponding SCA::Model::ContributionType class and provides additional information associated with a contribution. Examples for such additional information are:

  • the contribution's base URI
  • a collection of RuntimeModel::Composite instances contained within the contribution
